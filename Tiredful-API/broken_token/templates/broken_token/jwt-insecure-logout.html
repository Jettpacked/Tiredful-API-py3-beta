Challenge: JWT Token - Token Revocation

The scenario is to demonstrate the explicit logout/expiration of the JWT token. To invalidate the JWT token developer has to maintain the list of all tokens against which explicit invalidation request is initiated since the tokens will expire only after the set time period.

1. Following API is the improper implementation of JWT Token logout.

     GET  http://{{ request.get_host }}/api/v1/logout/


2. Following API is the secure implementation of JWT Token logout.
     GET  http://{{ request.get_host }}/api/v1/secure-logout/
